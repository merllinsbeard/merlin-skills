# Merlin Skills v0.1.2

Adds `install-merlin-skills`, a repo-owned meta skill for installing or refreshing this pack globally or project-locally.

## Included

- Added `skills/install-merlin-skills/SKILL.md`.
- Supported global Codex, global Claude, both global, project Codex, project Claude, and custom roots.
- Kept gstack runtime global by default at `~/.claude/skills/gstack`.
- Added project-local instruction block policy for `AGENTS.md` and `CLAUDE.md`.
- Updated routing, README, AGENTS, docs, report, changelog, and validation.

```bash
npm test
SKILL_ROOT=/tmp/merlin-skills-root GSTACK_ROOT=/tmp/merlin-gstack-runtime MERLIN_SKIP_GSTACK_BUILD=1 bash scripts/install-local.sh
```
