# Merlin Skills v0.1.5

Returns gstack to a minimal role in the Merlin stack.

## Changed

- Removed all gstack skills except adapted `gstack-qa` and `gstack-ship`.
- Deleted the `gstack-runtime` sidecar from the repo.
- Rewrote retained QA and ship skills as standalone Merlin gates for spec-kit/create-goal work.
- Updated installer cleanup so stale Merlin-managed gstack skills such as `gstack-review`, `gstack-office-hours`, and old unprefixed `review` are removed from the selected skill root when identifiable.
- Kept `gstack-qa` and `gstack-ship` prefixed on install.
- Updated routing, docs, manifest, notices, evals, and validation for 13 installable skills.
