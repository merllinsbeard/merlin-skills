# Merlin Skills v0.1.4

Namespaces gstack-derived installed skills.

## Changed

- Installs gstack-derived user-facing skills with the `gstack-` prefix, such as `gstack-review`, `gstack-qa`, `gstack-office-hours`, `gstack-codex`, and `gstack-plan-ceo-review`.
- Keeps the root `gstack` skill as the namespace entrypoint.
- Keeps `$GSTACK_ROOT` runtime directories unprefixed (`review/`, `qa/`, `office-hours/`, etc.) for upstream gstack compatibility.
- Patches installed gstack `name:` frontmatter and sets gstack `skill_prefix=true` during install.
- Updates `install-merlin-skills` and `merlin-skills-routing` so setup guidance and routing use install-facing names.
- Updates validation so routing coverage is checked against installed names, not just source snapshot directory names.
