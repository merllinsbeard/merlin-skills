# Merlin Skills v0.1.0

Initial open-source release.

## Included

- Full `github/spec-kit` source snapshot archive.
- Spec-kit-native `create-goal` skill adapted from AB Method's goal creation concept.
- Mandatory `merlin-skills-routing` meta skill for skill-first `/goal` execution.
- Selected Matt Pocock skills: `tdd`, `to-prd`, `grill-with-docs`, `diagnose`, `zoom-out`.
- Selected gstack skills: `review`, `qa`, `qa-only`, `ship`, design skills, and plan review skills.
- gstack compatibility runtime for upstream helper scripts and referenced runtime-only skills.
- Playwright browser proof layer: `playwright-cli` and `playwright-skill`.
- Superpowers `brainstorming` skill only.
- README cover generated with `imagegen` using `image-taste-frontend` art direction.
- Provenance manifest, third-party notices, validation scripts, installer, eval scaffold, and GitHub Actions workflow.
