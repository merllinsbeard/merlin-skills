# Merlin Skills v0.1.3

Fully integrates gstack while keeping Merlin Skills spec-first and `/goal`-first.

## Included

- Added all generic gstack skills, `codex`/`claude` wrappers, root `gstack` browser skill, and OpenClaw-specific gstack skills.
- Expanded `gstack-runtime` into a full curated runtime sidecar for browser, design, PDF, QA, review, docs, safety, GBrain, deploy, and upgrade helpers.
- Kept upstream gstack skill bodies verbatim.
- Kept live `SKILL.md` files out of `gstack-runtime`; the installer restores runtime skill files from `skills/<name>/SKILL.md`.
- Updated routing so full gstack is available, while the default workflow remains `spec-kit -> create-goal -> /goal`.
- Added validation that fails if any installed skill is missing from `merlin-skills-routing`.
